# etf-arb-sim
A simple simulation of ETF arbitrage opportunities using historical price data.
## Overview
This project simulates ETF arbitrage opportunities by analyzing historical price data of ETFs and their underlying assets. The simulation identifies potential arbitrage opportunities based on price discrepancies and calculates potential profits from executing arbitrage trades.
## Features
- Load historical price data for ETFs and their underlying assets.
- Identify arbitrage opportunities based on price discrepancies.
- Simulate arbitrage trades and calculate potential profits.
- Visualize price data and arbitrage opportunities using charts.
## Requirements
- Python 3.x
- pandas
- matplotlib
## Installation
1. Clone the repository:
   ```bash
   git clone
   ```
2. Navigate to the project directory:
   ```bash
   cd etf-arb-sim
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
## Usage
1. Prepare your historical price data in CSV format.
2. Update the configuration file with the paths to your data files.
3. Run the simulation script:
   ```bash
   python etf_arb_sim.py
   ```
4. View the results and visualizations generated by the simulation.
